Menús

En la parte superior de la interfaz se despliegan los siguientes menús que permiten acceder a las diferentes utilidades del programa:

  • Archivo
  • Ver
  • Flujo de trabajo
  • Herramientas
  • Ayuda

Archivo

Nuevo proyecto

Crea un nuevo proyecto vacio. En caso de que exista un proyecto o modelo abierto se cierran.

Método abreviado de teclado: Ctrl+N


Abrir proyecto

Se despliega una ventana que permite cargar un proyecto existente.

Método abreviado de teclado: Ctrl+O


Proyectos recientes

Se muestra un listado de los diez últimos proyectos abiertos permitiendo el acceso rápido a los mismos.

Desde el ménu Borrar historial se elimina el listado de ficheros recientes.

Guardar

Se guardan los cambios en el proyecto.

Método abreviado de teclado: Ctrl+S

Guardar como

Permite guardar con otro nombre el proyecto.

Método abreviado de teclado: Ctrl+Shift+S

Cerrar proyecto

Cierra el proyecto actual.

Salir

Cierra el programa.

Método abreviado de teclado: Ctrl+F4

Ver

Desde el menú Ver se activan o desactivan los paneles acoplables y las barras de herramientas.

Paneles acoplables

  • Proyecto: Abre el panel Árbol de proyecto. Si el panel esta abierto aparece resaltado con una marca de chequeo.
  • Propiedades: Abre el Propiedades. Si el panel esta abierto aparece resaltado con una marca de chequeo.
  • Miniaturas: Abre el Panel miniaturas. Si el panel esta abierto aparece resaltado con una marca de chequeo.
  • Consola: Abre la Consola. Si el panel esta abierto aparece resaltado con una marca de chequeo.

Barras de herramientas

Desde el menú Ver → Barras de herramientas se activan o desactivan las siguientes barras de herramientas:

  • Archivo
  • Visualización de imágenes
  • Flujo de trabajo
  • Modelo 3D
  • Herramientas

Flujo de trabajo


  • Añadir imágenes
  • Detección de puntos de interés
  • Correspondencia de puntos de interés
  • Orientación relativa

Herramientas

Cámaras

Herramienta que permite ver las cámaras del proyecto.

Visor de matches

Herramienta que permite la selección de pares de imagenes para ver las correspondencias encontradas entre sus puntos de interés.

Configuración de la aplicación

Ayuda

Ayuda del programa y ventana Acerca de... que muestra información de versión, autoría y licencias de terceros,

Método abreviado de teclado para la ayuda: F1
